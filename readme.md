# ERASERHEAD // Industrial Streetwear Store

![Main Banner](https://i.ibb.co/JwXw00zf/image.png)

### О проекте
Цифровая платформа для магазина одежды в стиле Cyber-Grunge. 
Проект включает в себя Full-stack реализацию: от управления товарами в админ-панели до фронтенда на React.

### Стек технологий
- **Frontend:** React, Tailwind CSS, Framer Motion
- **Backend:** ASP.NET Core API, Entity Framework
- **Database:** MySQL
- **Design:** Industrial / Opium Aesthetic

### Основные функции
- [x] Динамическая главная страница с видео-фоном
- [x] Админ-панель `VOID_CONTROL` для редактирования товаров
- [x] Система авторизации (JWT)
- [x] Адаптивная верстка
